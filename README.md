# Приложение с рецептами - Семестровая работа

Простое Android приложение для просмотра рецептов. 
Выполнено студентом КемГУ группы ФИТ-231 Сергиенко Александром Сергеевичем как семестровая работа по мобильной разработке.

## Что умеет приложение

- Показывает список рецептов с картинками
- Открывает детальную информацию о рецепте
- Работает с интернетом и без (кэширует данные)
- Имеет 2 экрана: список и детализация

## Технологии

- **Kotlin** - язык программирования
- **Jetpack Compose** - для создания интерфейса
- **Room** - локальная база данных
- **Retrofit** - для запросов к API
- **Coroutines** - для асинхронной работы

## Структура проекта

```
app/src/main/java/com/example/recipebook/
├── data/          # Работа с данными (база, сеть)
├── domain/        # Модели данных
├── ui/            # Экраны и ViewModel
└── MainActivity.kt
```

## Как работает

1. При запуске загружает рецепты из интернета
2. Сохраняет их в локальную базу данных
3. При следующем запуске показывает сохраненные данные
4. Обновляет данные когда есть интернет

## Запуск

1. Открыть в Android Studio
2. Синхронизировать Gradle
3. Запустить на эмуляторе или телефоне
